📋 テーブル構造

booth_id INT
PRIMARY KEY AUTO_INCREMENT
ブースを一意に識別するID
booth_name VARCHAR
NOT NULL
ブースの名前
description TEXT
NULLABLE
ブースの詳細説明
booth_type VARCHAR
NOT NULL
ブースのタイプ（game, shop）

🎯 ブースタイプ

🎮

ゲームブース

参加者がゲームを体験してステータスと所持金を獲得するブース

  • パワーチャージ・ステーション
  • リアクション・スプリンター
  • トレジャー・ハント
  • バーチャル・ランナー
  • ミラクル・タッチ

  • パワーアップアイテム
  • スタミナ回復アイテム
  • スピードブーストアイテム
  • テクニック向上アイテム
  • ラックアップアイテム

💡 使用例

ブース登録

INSERT INTO booths (booth_name, description, booth_type) 
VALUES ('パワーチャージ・ステーション', 'ボタン連打でパワーとスタミナを獲得', 'game');

ブース一覧取得

SELECT booth_id, booth_name, booth_type 
FROM booths 
ORDER BY booth_type, booth_name;

ゲームブースのみ取得

SELECT * FROM booths 
WHERE booth_type = 'game' 
ORDER BY booth_name;

ブース統計

SELECT booth_type, COUNT(*) as booth_count 
FROM booths 
GROUP BY booth_type;
